# Netflix-Analysis-Project
Power BI dashboard analyzing Netflix movies and TV shows by genre, release year, country, and duration. Explore trends, popular content, and insights interactively using the provided .pbix file.

## Getting Started  
1. Clone or download the repo.  
2. Open `netflix analytics dashboard.pbix` in Power BI Desktop.  
3. Explore dashboard visualizations — filter by genre, year, country etc., and inspect trends.

## Dataset  
Based on publicly available Netflix data (movies and TV shows), including details like title, genre, release year, duration, country, and rating.

## What You Get  
- Interactive Power BI dashboard with charts/graphs  
- Easy‑to‑browse content distribution by type, year, country
